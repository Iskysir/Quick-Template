介绍
================

本项目主要是一个快速泛型三层框架，使用MySQL作为数据库

很多公司采用三层结构，经常有很多项目，搭建框架也浪费不少时间，故此将本人多年常用的三层模式提炼出来.

此版本兼容Mysql 数据库，可以直接部署在Linux上。

注意：使用VS发布并不能把Repositories和Services一起发布到bin目录，需要手动切换Release然后生成拷贝到bin目录下。


主要技术

* MVC5

* AutoFac

* EntityFramework6

* log4net

* Mysql 数据库
